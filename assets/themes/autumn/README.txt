This is the theme folder for SPRING!

You can drop the following files here to customize it:

1. Music:
   - music.mp3 (Background music)

2. Background:
   - background.png (Main site background)
   - bottom.png (Decoration at the bottom of the page)

3. Decorations (Corners & Sides):
   - decoration_tl.png (Top Left)
   - decoration_tr.png (Top Right)
   - decoration_bl.png (Bottom Left)
   - decoration_br.png (Bottom Right)
   - decoration_top.png (Repeating top border)
   - decoration_left.png (Repeating left border)
   - decoration_right.png (Repeating right border)

4. Falling Particles:
   - falling_effect_1.png
   - falling_effect_2.png
   - ... (You can have as many as you want, just set "falling_effect_count" in config.js)
